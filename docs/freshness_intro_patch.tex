The spine of this project is a single objective: \emph{energy-optimal semantic
knowledge aggregation for robot/drone swarms tracking \textbf{moving} targets},
in which target motion enters not through physical chasing --- which we verify is
inert in this framework --- but through \textbf{information freshness}, the value
of a report that decays as the observed target moves. The two recommended
directions are the two mechanisms that realize this objective: \Dir{1} makes the
report's fidelity measurable and freshness-aware (so the compression ratio bends
to target motion), and \Dir{2} makes the aggregation topology value-prioritized
(so fast-decaying sources reach the root on shorter paths). Section~\ref{sec:freshness}
reports the implemented spine: $58$ automated checks, $0$ failures, and an
interactive demo.
